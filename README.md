# Basketball Duck Pet

Standalone Windows desktop pet built from the Basketball Duck spritesheet.

## Development

```powershell
npm install
npm start
```

## Build Installer

```powershell
npm run dist
```

The installer will be written to `release/BasketballDuckPet-Setup-1.0.0.exe`.

## Controls

- Left click: wave
- Double click: jump
- Drag: move the pet around the desktop
- Right click: open the menu
- Tray icon: show or hide the pet
